Team Nova Portfolio

Team Nova is a university portfolio project created to showcase the creativity, technical skills, and professional abilities of our four team members. This platform provides visitors with an interactive experience where they can explore each member's role, expertise, and personal portfolio.

Project Goals

  • Create a modern team portfolio platform
  • Showcase individual member skills and achievements
  • Provide easy access to personal GitHub portfolios
  • Demonstrate frontend development skills

Technologies

HTML5 CSS3 JavaScript Responsive Design UI/UX Design
